# FillDoc (MVP / Этап 1)

Десктопная программа для автоматического заполнения юридических документов по шаблонам Word (`.docx`) на основе карточек проектов из Excel.

## Запуск (Windows)

1) Установить зависимости:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

2) Запустить приложение:

```bash
python main.py
```

## Ограничения MVP

- Поддерживается только `.docx` (без `.doc`).
- Источник проектов — Excel-файл.
- Библиотека шаблонов — папка на локальном диске с подпапками.
- Синхронизация только по действиям пользователя (Открыть/Обновить/Сохранить).
- Без многопользовательского режима, без фоновой синхронизации, без истории изменений, без предпросмотра документа.

# Формат Excel-файла

FillDoc использует Excel как простую базу проектов. Один проект соответствует одной строке таблицы.

## Поддерживаемые файлы

- `.xlsx`;
- `.xlsm`.

Старые `.xls` не являются целевым форматом проекта.

## Листы

### `Текущие`

Основной лист с активными проектами. Если листа `Текущие` нет, FillDoc использует первый лист книги.

### `Архив`

Опциональный лист для архивных проектов. Если архив нужен, его заголовки должны совпадать с заголовками основного листа. Приложение умеет восстанавливать или обновлять заголовки архива при некоторых операциях.

## Строка заголовков

Первая строка листа должна содержать названия полей:

```text
№ дела | Заказчик | Должник | ИНН должника | ОГРН должника | Дата решения
```

Заголовки не должны быть пустыми для тех колонок, которые нужны в шаблонах.

## Строки проектов

Начиная со второй строки:

- каждая непустая строка считается проектом;
- полностью пустые строки игнорируются;
- строка, похожая на повтор заголовков, игнорируется;
- значения ячеек читаются как строки.

## Идентификатор проекта

FillDoc пытается взять идентификатор из колонки номера дела. Поддерживаются варианты:

```text
№ дела
№дела
Номер дела
Номер осн. дела
Номер осн дела
```

Если подходящего значения нет, используется служебный идентификатор:

```text
row:<номер строки>
```

Например:

```text
row:7
```

## Связь с Word-шаблонами

Самый надежный вариант - использовать одинаковые имена:

Excel:

```text
Должник
```

Word:

```text
{Должник}
```

Если в шаблоне используется вариант имени, словарь переменных может сопоставить его с основной колонкой. Но для поддержки проекта проще постепенно приводить Excel и шаблоны к одному набору названий.

## Запись и бэкапы

Перед изменением Excel FillDoc создает резервную копию рядом с исходным файлом:

```text
_filldoc_backups\<имя файла>__backup__YYYYMMDD_HHMMSS.xlsx
```

Текущая логика ротации оставляет ограниченное количество свежих копий и удаляет устаревшие.

## Рекомендации

- Не объединяйте ячейки в строке заголовков.
- Не оставляйте две разные колонки с одинаковым названием.
- Не меняйте названия колонок без одновременного обновления шаблонов.
- Закрывайте Excel перед сохранением изменений из FillDoc.
- Держите архивный лист в той же структуре, что и основной.

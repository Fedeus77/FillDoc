# Формат Word-шаблонов

FillDoc работает с Word-файлами `.docx` и заменяет переменные на значения из проекта.

## Где хранить шаблоны

В настройках приложения указывается корневая папка шаблонов. FillDoc сканирует ее рекурсивно:

```text
templates\
  Акты\
    Акт.docx
  Заявления\
    Заявление.docx
```

Название подпапки используется как категория шаблона в интерфейсе.

## Формат переменной

Переменная - это текст в фигурных скобках:

```text
{Имя поля}
```

Примеры:

```text
{Должник}
{Заказчик}
{ИНН должника}
{Дата решения}
```

Во время генерации FillDoc ищет плейсхолдер целиком, включая скобки, и заменяет его на значение.

## Где поддерживаются переменные

Поддерживаются:

- обычные абзацы;
- таблицы;
- вложенные таблицы;
- верхние и нижние колонтитулы;
- колонтитулы первой, четной и нечетной страницы.

## Имена переменных

Рекомендуемый стандарт: имя переменной в Word совпадает с названием колонки Excel.

Excel:

```text
ИНН должника
```

Word:

```text
{ИНН должника}
```

Если имя отличается, FillDoc может попробовать сопоставить его через словарь переменных, но это не должно быть основным способом поддержки шаблонов.

## Правила оформления

- Используйте одинарные фигурные скобки: `{Должник}`.
- Не используйте вложенные скобки: `{{Должник}}`.
- Не добавляйте лишние пробелы без необходимости: `{ Должник }` будет считаться именем `Должник` при извлечении, но лучше писать единообразно.
- Не используйте старый формат `.doc`.
- Не редактируйте временные файлы Word, которые начинаются с `~$`.

## Кэш шаблонов

При сканировании FillDoc создает служебную папку:

```text
<папка шаблонов>\.filldoc
```

В ней лежат JSON-карточки шаблонов. Они хранят найденные переменные и правило имени выходного файла. Эту папку можно удалить: при следующем сканировании она будет создана заново, но ручные настройки карточек могут быть потеряны.

## Имена выходных файлов

Имя готового файла очищается от символов, запрещенных Windows:

```text
< > : " / \ | ? *
```

Если файл с таким именем уже есть, FillDoc сохраняет следующий вариант с номером:

```text
Документ.docx
Документ (2).docx
Документ (3).docx
```
